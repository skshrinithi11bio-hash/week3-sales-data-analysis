# Sales Data Analysis Report

## Objective
To analyze a simple sales dataset and extract key business insights such as total revenue, average sales, and best-selling product.

## Analysis Steps
1. Loaded the dataset using pandas.
2. Checked dataset structure and identified missing values.
3. Filled missing values using mean imputation.
4. Calculated total sales for each record.
5. Derived summary metrics.

## Key Findings
- Total revenue was calculated across all products.
- Average sales value was determined.
- Laptop emerged as the best-selling product based on total revenue.

## Conclusion
The analysis provides a clear understanding of sales performance and highlights top-performing products using basic data analysis techniques.
